Free bot for giveaway sites.

**Support**
* GameMiner
* SteamGifts
* SteamCompanion
* SteamPortal
* UseGamble
* PlayBlink
* GameAways
* InventoryGifts

**Minimum system requirements**: Windows 7 and .NET Framework 4.5
